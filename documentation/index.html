Car Listing

CarSpace - Car Listing Directory with Subscription System

Created on: August 18, 2019

By: GeniusOcean

Email: badruddoza.me@gmail.com

About

“Car Listing” is a subscription based classified site for cars. It has 2 panels - Admin and seller panel. Here Admin will create some plans and seller have to buy those packages to post ads. Different plans allows sellers to post different numbers of ads. This plan also has an expiration date. If you want to start a car classified site then this is the perfect site to launch your business.

Installation

After downloading file from codecanoyn just extract in your computer and you will get three folders:

Step 1: Extract files.zip file from Files folder to server.

Step 2: Create database using database.sql from Database folder.

Step 3: Edit details of .env file in project folder.

To login into admin panel, visit:
yourwebsiteURL.com/admin
User: admin@gmail.com
Password: 1234

Cron job

You need to set cron job to check package validity everyday. if the expiriration date is over then user will be notified via email also users will get an alert before 5 days of expiration date of their package.
Cron Command: wget http://YourDomain.Name/checkvalidity
Cron Time: Set the time to 'once per day' to run the validation check daily.

Admin:

You can change each and everything of this system from this Admin panel. From general settings you can change the basic settings of this site such as colors, logo, favicon etc. You can also update the contents of pages. From email settings you can change informations of Email. Form language settings you can change the language of this site. You can also make it SEO friendly using the SEO tools section of admin panel. You can also post articles from admin panel.

Dashboard:

Here admin can check few basic statistics of his/her agency.

Car Specifications

In Car Specifications, Admin can manage all the required car specifications.

Category Management: Admin can manage all the category of cars.

Condition Management: Admin manage all the conditions of cars.

Brand Management: Admin manage all the brands of cars.

Model Management: Admin manage all the models of cars.

Body Type Management: Admin manage all the body type of cars.

Fuel Type Management: Admin manage all the fuel type of cars.

Tranmission Type Management: Admin manage all the transmission type of cars.

Pricing Ranges:

Admin can setup all the priceing ranges for search.

Plan Management:

Here admin can create, delete, update different plans of different prices and different expiration dates.

Car Management

In Car management section, Admin can edit delete all cars. These cars are kept in different sections such us featured, published, pending, rejected cars etc.

All Cars: Admin can manage all types of cars.

Featured Cars: Admin can manage the cars which have been featured by Admin.

Pending Cars: Admin can manage the cars which have not been published / rejected yet.

Published Cars: Admin can manage the cars which have not been published.

Rejected Cars: Admin can manage the cars which have not been rejected.

Sellers Management:

Here admin can active/deactive sellers and also can change their informations.

General Settings

In General Settings, Admin can add/edit logo, favicon, loader, background image, review background image etc.

Logo: Admin can upload logo from here. Logo width should be 280px or less.

Favicon: Admin can upload Favicon from here. Favicon size should be 32x32px.

Loader: Admin can upload loader from here.

Breadcrumb: Admin can upload background image for breadcrumb of every page from here.

Website Contents: Here admin can set website title which will show on browser tab. Admin can also change Theme color of the system. Also admin can set tawk.to live chat and disqus comment system for blogs.

Payment Informations: Here admin can set client id of paypal account and secret key of stripe account.

Footer: Here admin can change footer text, footer logo, copyright text.

Social Settings: Here admin can change social profile links and hide / show specific social profile.

Home Page Settings

In Home Page Settings, Admin update each and every section of home page.

Header Banner: Admin can update background image and texts of header section.

Featured Cars Section: Admin can update texts of featured cars section.

Latest Cars Section: Admin can update texts of latest cars section.

Testimonial Management: Admin can create, update, delete testimonial of clients.

Blog Section: Admin can update texts of blog section.

Menu Page Settings

In this section admin can control the FAQ , Contact Us page and other pages. You can use the editor to write your content and decorate it with the HTML builder. You can add unlimited pages from the Others Pages section. You can also set where the page will available in header or in footer.

Email Settings

Email Template: Admin can design his own email template from here. He can add the dynamic fields and any elements from the text editor. If admin wants to mention the customer name then just copy the BB CODE{customer_name} from the above table and paste in your template. It will automatically identify the client name.

Email Configuration: Admin can setup SMTP or PHP mail service from here. If admin deactivate the SMTP from top section then the website email system will be PHP mail system. Your hosting provider can help you to fill up the form.

Group Email: Admin can send group emails from this option. Just select your options then write your mail and send.

Language Settings

In Language Settings, Admin update each and every word in his / her native language.

Blog:

Admin can manage blog from here. To Add Blog post admin will have to click on Add New Blog from the top right corne. Admin have to complete the simple steps from Add New Blog page. Please use same size image for all the blog post, So it blog images will be same in any device like mobile, computer or tablet computer. Admin can Edit and Remove added blog by single click.

SEO Tools:

Admin can check websites most popular products from here. Admin can filter it too. From here admin can setup google analytics and meta keywords. It is mandatory field for any website for better SEO result and checking the traffic.

Payment History:

Admin can see all the payment history.
